# malakasisn.github.io

Personal website for [Nikos Malakasis](https://malakasisn.github.io), hosted on GitHub Pages.

## Stack

Plain HTML, CSS, and vanilla JS — no build step, no frameworks.

## Structure

```
index.html              # Single-page site
style.css               # All styles (CSS custom properties, responsive)
script.js               # Minimal JS (nav scroll effect, mobile toggle)
Nikos_Malakasis_CV.pdf  # CV linked from the site
.nojekyll               # Disables GitHub Pages Jekyll processing
```

## Local preview

Open `index.html` directly in a browser — no server needed.

## Deployment

Pushing to `main` deploys automatically via GitHub Pages.
